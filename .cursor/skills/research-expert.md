# Research Expert Skill

## Purpose
Exhaustive information gathering and analysis before any proposal or change.

## When to Trigger
- When starting any task.
- When encountering an unknown error.
- When exploring a new part of the codebase.

## Research Strategy
1. **Breadth-First**: List directories to understand the structure (`list_dir`).
2. **Depth-First**: Read key files and search for specific patterns (`read_file`, `grep_search`).
3. **External Discovery**: Use the `browser` or `search_web` to find documentation or solutions for libraries used.
4. **Synthesis**: Combine all findings into a clear summary of how things currently work.

## Deliverables
Detailed descriptions of the current state, including file paths, model schemas, and logic flows.
Propose changes ONLY after this research is complete.
